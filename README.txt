SuzzyTech - Free Unlimited AI-Generated Video Gate
-----------------------------------------------

Files included:
- index.html    (single-page mobile-first landing)
- styles.css
- script.js
- assets/logo.svg, assets/favicon.png

Important:
- The verification method is time-locked client-side (not a secure server-verified gate).
- Replace assets/logo.svg with your own logo for branding.
- You can host this folder on GitHub and deploy to Vercel or Netlify.
- To fully verify YouTube subscriptions, integrate YouTube API server-side (not included).

How to deploy:
1. Upload the folder to a GitHub repo.
2. On Vercel, import the repo and deploy. Or enable GitHub Pages and serve index.html.
3. Update links or change the prompt text inside index.html if needed.
